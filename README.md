### [时序差分算法可视化](/demos/temporal_difference_visualization.html)
一个综合的时序差分算法分析平台，支持四种算法的实时对比：

#### 🔬 四算法综合实验
- **原始Sarsa算法**：固定参数的标准在线策略实现
- **改进Sarsa算法**：参数衰减优化版本，解决策略波动问题
- **原始Q-learning算法**：固定参数的标准离线策略实现  
- **改进Q-learning算法**：参数衰减优化版本，确保稳定收敛

#### 🎯 核心分析功能
- **八条学习曲线对比**：原始数据 + 移动平均的完整对比展示
- **参数衰减可视化**：实时显示ε和α值的动态变化过程，支持长期持续衰减
- **最优策略追踪**：记录达到最优策略的具体回合数
- **深度收敛分析**：Q值变化、更新幅度、稳定性指标的多维度分析

#### 🛠️ 交互控制特性
- **防误操作保护**：训练期间自动禁用操作按钮，避免重复执行
- **异步分批训练**：大批次训练时保持界面响应性
- **单步调试模式**：细粒度观察每回合的参数变化
- **一键重置功能**：快速重新开始对比实验，参数正确重置为0.1
- **参数重置修复**：解决了重置按钮点击时学习率和探索率无法正确重置的问题
- **智能续训机制**：继续训练时参数从上次结束位置继续衰减，实现真正的续训功能
- **参数衰减追踪**：实时显示ε和α的衰减百分比，直观展示参数变化趋势
- **长期衰减优化**：参数最小值设为0.001，衰减系数0.998，确保2000+回合的持续衰减
- **高精度显示**：参数显示精度提升至5位小数，清晰展示微小变化
- **输入框稳定性**：输入框保持用户设定的原始值，不随训练过程自动更新
- **参数分离显示**：输入框显示用户配置值，算法内部参数变化通过状态信息展示
- **训练状态区分**：自动识别首次训练vs继续训练，确保参数处理逻辑正确
- **现代化UI设计**：全面重构页面视觉设计，采用渐变背景、毛玻璃效果和流畅动画
- **酷炫交互体验**：悬停特效、闪光按钮、网格动画等增强用户交互反馈
- **优雅视觉层次**：统一的配色方案、现代化卡片设计和专业图标系统

#### 📊 学习率分析深度功能
- **Q值收敛过程监控**：跟踪特定状态Q值的完整学习轨迹
- **后期收敛细节分析**：专注最后100回合的精细收敛过程
- **学习率衰减对比**：固定学习率vs衰减学习率的效果对比
- **收敛稳定性量化**：通过更新幅度标准差评估算法稳定性

#### 🎨 现代化UI设计特性
- **渐变背景系统**：采用Purple到Blue的优雅渐变背景，营造科技感氛围
- **毛玻璃效果**：所有卡片采用半透明毛玻璃设计，配合backdrop-filter模糊效果
- **动态交互动画**：
  - 页面加载动画：fade-in、slide-up、scale-in等流畅进入效果
  - 悬停动画：网格、按钮、卡片的精致悬停效果和阴影变化
  - 闪光特效：按钮点击时的光束扫过效果
- **现代化配色方案**：
  - 渐变按钮：Success绿色、Primary紫色、Danger红色的渐变设计
  - 彩色指示器：每个输入项配备独特的渐变色彩标识
  - 阴影系统：多层次阴影营造空间深度感
- **交互反馈系统**：
  - 按钮悬停时的上浮效果和阴影增强
  - 网格单元格的放大和闪光效果
  - 图例项的缩放和背景高亮
- **现代化图标系统**：每个功能区域配备SVG图标，提升视觉识别度
- **优化文字可读性**：
  - 网格单元格采用高对比度配色方案，确保状态值清晰可见
  - 特殊单元格（起点/终点/悬崖）使用白色文字配阴影效果
  - 普通单元格使用深色文字配浅色渐变背景

  - 最后20回合奖励对比表格按算法类型分组
  - Sarsa与改进Sarsa相邻，Q-learning与改进Q-learning相邻
  - 便于横向对比原始算法与其改进版本的性能差异

## 🚀 快速开始

### 本地运行
```bash
# 安装依赖
bundle install

# 启动本地服务器
bundle exec jekyll serve --livereload
```

### 项目结构
```shell
.
├── _config.yml          # Jekyll 配置文件
├── _posts/              # 博客文章
├── _tabs/               # 导航页面
├── demos/               # 演示与可视化工具
│   └── cliff_walking_visualization.html
└── assets/              # 静态资源
```
